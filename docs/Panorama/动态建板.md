# snippet

模版，便于在js中直接创建组合板。

## 范例

```xml
	<snippets>
		<snippet name="QuestLine">

			<Panel><!--注：创建板时，赋名的是这一层的板-->
				<Panel direction="horizontal" class="Progress">
                    <Label class="slider_info" text=""/>
				</Panel>

				<Panel class="slider_lable">
					<Label class="slider_info" text=""/>
				</Panel>

			</Panel>
		</snippet>
	</snippets>
```

动态创建板
```js
//
var NewPanel = $.CreatePanel('Panel', $('#QuestExample'),'QuestPanel');//在QuestExample下创建一个新的 panel，id赋值为 QuestPanel
    NewPanel.BLoadLayoutSnippet("QuestLine");//加载我们的模版
    
```

删除这个板
```js
//
$('#QuestExample').FindChild('QuestPanel').DeleteAsync(0);
    
```

# BCreateChildren

通过在父板上创建子板的方法也可以动态建板，也更灵活

```js
$('#QuestExample').BCreateChildren("<Panel id='QuestPanel' style='flow-children:left;' />" );
$('#QuestPanel').BCreateChildren('<Panel direction="horizontal" class="Progress"/>' );
$('#QuestPanel').GetChild(0).BCreateChildren('<Label class="slider_info" text=""/>' );
```
